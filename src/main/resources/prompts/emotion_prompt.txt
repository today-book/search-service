아래 문장에 대해 다음 두 값을 정확한 JSON 형식으로만 반환하세요.

   1) emotion:
      - 문장의 감정을 ["기쁨","슬픔","불안","분노","힐링","동기부여","중립"] 중 하나로 선택
      - 반드시 목록 중 하나만 사용

   2) query:
      - 임베딩 검색에 최적화된 한 문장을 생성
      - 원본 의미(질의 의도, 분위기, 성향 포함)를 유지
      - 감정(emotion)과 자연스럽게 결합된 형태로 재작성
      - 문장 길이는 너무 길거나 장황하지 않게
      - 한 문장만 생성
      - 불필요한 말 제거
      - 문장의 마지막 단어는 반드시 "책" 또는 "책." 이어야 함

입력 문장: {query}

⚠️ 출력 규칙 (매우 중요)
- 반드시 JSON ONLY 출력 (설명·문장·접두사·코드블록 금지)
- 출력 JSON은 다음 두 key만 포함: "emotion", "query"
- 예시 형식:
  \{"emotion": "불안", "query": "불안한 마음을 달랠 수 있는 차분한 분위기의 책"\}
- null, 빈 문자열, 추가 필드 절대 금지
- key 순서 유지: emotion → query
