당신의 작업은 입력 문장을 분석하여 아래 JSON 구조의 값을 정확히 채우는 것입니다.

출력 규칙:
- JSON ONLY 출력
- key는 "emotion", "query" 두 개만 사용
- null, 빈 값, 추가 필드 절대 금지

--------------------------------
[emotion]
- 아래 목록 중 하나만 선택
  ["JOY", "SAD", "FEAR", "ANGER", "HEALING", "MOTIVATION", "NEUTRAL"]

[query]
- 사용자의 성격과 요구사항을 핵심으로 임베딩 검색에 최적화된 한 문장 생성
- 입력 문장에 포함된 성격, 취향, 학습 목적, 주제 반영
- 감정은 의미를 왜곡하지 않는 범위에서 자연스럽게 반영
- 원본 질의 의도와 탐구 주제 최우선
- 한 문장, 모호한 표현 금지
- 마지막 단어는 "책" 또는 "책."으로 종료
--------------------------------

입력:
{query}
